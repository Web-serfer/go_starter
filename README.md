# Go Game Application

Это приложение на языке Go, представляющее собой географический квиз, в котором игроку предлагается угадать столицы различных стран.

## Описание

Проект представляет собой консольную игру-викторину, написанную на языке программирования Go. Игроку предлагается 10 вопросов о столицах стран, с несколькими попытками на каждый вопрос. В игре реализована система жизней и подсчета очков. Также доступна функция подсказки при затруднении.

Основная логика игры находится в пакете `internal/game`, а точка входа в приложение расположена в `cmd/app/main.go`.

## Особенности игры

- 10 вопросов о столицах разных стран
- 3 жизни на протяжении всей игры
- 4 попытки на каждый вопрос
- Система подсказок (доступна после 2 ошибок на один вопрос)
- Возможность сдаться и получить правильный ответ
- Интересные факты о столицах
- Подсчет очков в зависимости от количества использованных попыток

## Запуск приложения

Для запуска приложения выполните следующую команду:

```bash
go run cmd/app/main.go
```

Либо соберите бинарный файл и запустите его:

```bash
go build -o app cmd/app/main.go
./app
```

## Игровой процесс

1. При запуске игры вы получите приветствие и предложение начать игру
2. На каждый вопрос у вас есть 4 попытки
3. После 2 ошибок на один вопрос становится доступна подсказка
4. Для получения подсказки введите "подсказка" или "hint"
5. Если вы не знаете ответ, можно ввести "не знаю", "сдаюсь" или "give up"
6. За правильный ответ начисляются очки (чем меньше попыток использовано, тем больше очков)
7. Игра продолжается до окончания вопросов или до потери всех жизней
